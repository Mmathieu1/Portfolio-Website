How Cheesy Are You?

Personality Quizes, an enjoyable time-filler in our day that can result in some humorous results that we 'discover' about ourselves. Whether it's inner animal spirit or what kind of shoe we would be, there's a personality quiz for everyone. This personality test is a one person project created during CarrerFoundry's Voice Interaction with Alexa course. During this project I created user flows, user scripts, questions and results while editing a given base code to customize Alexa's reponses and phrases. This skill is also designed to take advantage of Alexa Echo's visual screen ability-using fun images throught the quiz.

Overview

'How Cheesy Are You?' is a personality quiz that has taken the lighthearted approach. A fun, quick and humorous quiz for those who are cheese-curious about themselves and where they could enjoy a Gouda Pun as well.

Approach

Designing for the Ear * User Centered Design * Design Thinking
